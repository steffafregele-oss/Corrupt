# Discord Stats Bot

Bot Discord care afișează statistici din API-ul Injuries.lu

## Comenzi
- `!stats` — Arată statisticile tale
- `!stats @user` — Arată statisticile altui utilizator
- `!daily` — Arată statisticile zilnice

## Host gratuit
Poți hosta botul pe **Railway** gratuit:
1. Mergi pe https://railway.app
2. Fă login cu GitHub
3. Click “New Project” → “Deploy from GitHub repo”
4. Selectează repository-ul tău
5. Adaugă `Environment Variable`:  
   - **Key:** `DISCORD_BOT_TOKEN`  
   - **Value:** tokenul botului tău din [Discord Developer Portal](https://discord.com/developers/applications)
6. Apasă **Deploy**

Botul tău va fi online 24/7 ✅
